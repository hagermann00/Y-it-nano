## **Search Strategy Tips**

### Multi-Platform Approach
1. **Unsplash**: Best for high-quality, modern aesthetic photos
2. **Pexels**: Good for business/office imagery
3. **Pixabay**: Larger volume, more variety (but variable quality)

### Search Refinement
- Start broad, then narrow: "warehouse" → "empty warehouse" → "abandoned warehouse interior"
- Try synonyms: "entrepreneur" / "business owner" / "startup founder"
- Use modifiers: "modern", "minimalist", "professional", "realistic"

### Quality Filters
- Sort by: "Popular" or "Curated" (higher quality)
- Orientation: "Landscape" for web, "Portrait" for book covers
- Size: "Large" (minimum 1920px width)

### License Verification
- Always verify CC0 license on each image
- Check "Free for commercial use" / "No attribution required"
- Download from official source (not third-party)
- Save license info in metadata

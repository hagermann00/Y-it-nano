## RISK MITIGATION: BOOTSTRAPPING EDITION

### Top 7 Bootstrapping Risks

1. **Free tier limits exceeded before revenue** (Probability: HIGH)
   - **Mitigation:** Monitor usage weekly, optimize queries, cache aggressively
   - **Contingency:** Pause new topic launches, optimize existing topics first
   - **Cost:** Supabase Pro = $25/mo, only upgrade when revenue > $250/mo

2. **DIY design quality too low, poor reviews** (Probability: MEDIUM)
   - **Mitigation:** Invest 20 hrs in master template, get 3 beta readers
   - **Contingency:** Hire designer for $200 to fix first book, reuse template
   - **Prevention:** Set quality bar: 4.0+ Amazon rating or pull & redo

3. **Organic marketing doesn't generate 1,000 visitors/topic** (Probability: MEDIUM-HIGH)
   - **Mitigation:** Multi-channel approach (Reddit + SEO + Product Hunt + podcasts)
   - **Contingency:** Spend $100-200 on Facebook ads for validation (one-time)
   - **Pivot:** If organic fails after Topic 3, reassess content or distribution

4. **3% conversion, can't get to 5%** (Probability: MEDIUM)
   - **Mitigation:** 2-field form, social proof, email optimization, pricing tests
   - **Contingency:** Focus on AOV (bundles), recurring revenue (subscriptions)
   - **Reality check:** If still 3% after 10 topics, model doesn't work

5. **Burnout from 150 hrs/month + full-time job** (Probability: HIGH)
   - **Mitigation:** Batch content (1 topic per month), automate marketing
   - **Contingency:** Slow to 1 topic per 2 months (50 topics = 100 months)
   - **Prevention:** Hire VA for $300/mo when revenue > $3,000/mo

6. **Platform complexity too high, can't code it yourself** (Probability: MEDIUM)
   - **Mitigation:** Use no-code first (Carrd + Gumroad + Zapier), validate, then code
   - **Contingency:** Pay $2,000 for developer to build MVP (breaks budget, but salvages project)
   - **Alternative:** Partner with technical co-founder (give 20-30% equity)

7. **Legal issues (copyright, tax, liability)** (Probability: LOW)
   - **Mitigation:** Use free KDP ISBNs, file LLC ($100), get EIN (free), use LegalZoom free articles
   - **Contingency:** Pause sales, consult lawyer ($300), fix compliance
   - **Prevention:** Anonymize all case studies, use disclaimers, don't make medical/financial advice claims

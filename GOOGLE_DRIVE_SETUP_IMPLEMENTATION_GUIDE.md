## 🤖 AUTOMATION SCRIPT (Optional)

### Google Apps Script to Auto-Create Folders

```javascript
// Copy this into Google Apps Script editor
// Create: Extensions → Apps Script → New Project

function createBatchFolders() {
  const rootFolderId = "YOUR_ROOT_FOLDER_ID"; // Get from URL
  const rootFolder = DriveApp.getFolderById(rootFolderId);

  // Define batches and topics
  const batches = {
    'Batch A': ['Dropshipping', 'Amazon FBA', 'Print-on-Demand', 'Affiliate Marketing',
                'Course Creation', 'SMMA', 'YouTube Monetization'],
    'Batch B': ['Freelance Writing', 'Virtual Assistant Service', 'Shopify Store',
                'Content Syndication', 'Twitter Monetization', 'TikTok Creator Fund',
                'Etsy Shop', 'OnlyFans Creator', 'Twitch Streaming', 'Medium Writing'],
    // ... add remaining batches
  };

  // Create folder structure
  for (let batchName in batches) {
    const batchFolder = rootFolder.createFolder(batchName);

    for (let topic of batches[batchName]) {
      const topicFolder = batchFolder.createFolder(topic);

      // Create subfolders
      topicFolder.createFolder('01_RESEARCH');
      topicFolder.createFolder('02_CONTENT');
      topicFolder.createFolder('03_DESIGN');
      topicFolder.createFolder('04_MARKETING');

      Logger.log(`Created: ${batchName}/${topic}`);
    }
  }

  Logger.log('Folder structure created successfully!');
}
```

**How to use:**
1. Open Google Drive
2. Right-click → "Create new" → "Google Apps Script"
3. Copy script above
4. Replace `YOUR_ROOT_FOLDER_ID` with your root folder ID
5. Click "Run"
6. Authorize when prompted
7. Folders created automatically in ~5 minutes

## 📊 SPREADSHEET FORMULAS & SETUP

### Affiliate Score Calculation (Google Sheets)

```
=N2+O2+P2

Where:
N = Commission Score (0-2)
O = Conversion Score (0-2)
P = Relevance Score (0-2)

Plus optional:
+ IF(D2="Easy", 0.5, 0) for difficulty bonus
+ IF(D2="Approved", 1, 0) for approval bonus

Max Score = 10 points
```

### Revenue Projection Formula

```
=A2 * B2 * C2 * D2 * 12

Where:
A = Monthly customers
B = Affiliate CTR (click-through rate)
C = Conversion rate
D = Average commission per sale
* 12 = Annual projection
```

### Progress Tracking Formula

```
=COUNTIF(A:A, "Complete") / COUNTA(A:A) * 100

Shows: (Completed topics) ÷ (Total topics) = % complete
```

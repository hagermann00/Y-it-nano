## UNIVERSAL DESIGN SYSTEM SPECS

### Page Layout
- **Trim Size:** 6.000" × 9.000"
- **Bleed:** 0.125" on all sides
- **Safe Margin:** 0.25" from trim edge
- **Text Block:** 5.5" × 8.25" (centered)
- **Gutter:** 0.5"

### Color System
- **Background:** White (#FFFFFF)
- **Body Text:** Dark Gray (#333333)
- **Headers:** Book-specific accent color (see above)
- **Callout Boxes:** 5% tint of accent color background
- **Pull Quotes:** Book-specific accent color, italicized

### Font Specifications
- **Headings (H1-H3):** Bebas Neue (Google Fonts - free)
- **Body Text:** Lato (Google Fonts - free)
- **Code/Data:** Open Sans Mono (Google Fonts - free)
- **Fallback:** If Bebas Neue unavailable, use Oswald; if Lato unavailable, use Source Sans Pro

### Image Specifications
- **Resolution:** 300 DPI minimum (for print quality)
- **Format:** PNG (preferred) or JPG (acceptable)
- **Charts/Graphs:** Prefer vector graphics (SVG) for scalability
- **Color Space:** RGB, will convert to CMYK for printing
- **Dimensions:** Full-bleed images are 6.25" × 9.25"; contained images 5.5" × varies

### Callout Box Design
```
┌─────────────────────────────────────────┐
│ CALLOUT TITLE (Bebas Neue, accent color) │
├─────────────────────────────────────────┤
│ Content in Lato, dark gray, 1.5 spacing │
│ with subtle 5% tint background          │
└─────────────────────────────────────────┘
```

### Data Table Design
- **Header Row:** 25% gray background, white text, Lato Bold
- **Data Rows:** Alternate white/light gray (2% tint)
- **Borders:** Light gray (#EEEEEE)
- **Font:** Lato, 10pt, dark gray

### Pull Quote Design
```
        ❝ Key statistic or insight quote
            - Source or context ❞
```
Formatting: Lato 14pt italic, accent color, indented 0.25"

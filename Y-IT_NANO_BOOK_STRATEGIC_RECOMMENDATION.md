## 1. FORMAT RECOMMENDATION: 6×9" STANDARD TRADE PAPERBACK

### Decision Rationale

| Factor | 6×9" | 8.5×11" | Winner |
|--------|------|---------|--------|
| **Print COGS (12 pages)** | $1.50-1.85 | $2.00-2.75 | 6×9" |
| **$2.99 margin after Amazon** | ~$0.85 | ~$0.35 | 6×9" ✓ |
| **Perceived credibility** | Professional book | Workbook/DIY | 6×9" ✓ |
| **Portability** | Jacket pocket | Must carry | 6×9" ✓ |
| **Industry category fit** | Business/self-help | Technical manual | 6×9" ✓ |
| **Shelf presence** | Standard bookshelf | Coffee table/desk | 6×9" ✓ |
| **Chart readability** | 11pt body = tight | 12-13pt = readable | 8.5×11" |
| **Visual breathing room** | Conservative | Generous | 8.5×11" |

### Financial Justification
- **At $2.99 KDP:** 6×9" = $0.85 net/copy; 8.5×11" = $0.35 net/copy
- **At 1,000 copies/month:** 6×9" = $850/month; 8.5×11" = $350/month
- **Bundle option:** Print + Digital at $4.99 = higher acceptable COGS
- **Psychological pricing:** "$2.99 book" feels like complete product; "$2.99 pamphlet" feels incomplete

### Reader Psychology
Your target audience (failed entrepreneurs, due diligence researchers) expects business book format. 8.5×11" triggers "workbook" associations which paradoxically devalue the $2.99 price point (feels cheap). 6×9" says "published business analysis" which justifies premium pricing in bundles.

### Design Advantage
6×9" forces focused design. Limits images to 6-8, which means each image must earn its space. No visual padding = tighter, more impactful layout. Readers perceive density as credibility.

**✓ DECISION: 6×9" format with 12-page target**

3. **Digital advantage:** Animated comic > static panels (justifies web upgrade)
4. **Production timeline:** 1 quality illustration (1 week) vs 4-6 comic panels (2-3 weeks)
5. **Scaling:** Same hero image works across all 7 Y-IT nano-books

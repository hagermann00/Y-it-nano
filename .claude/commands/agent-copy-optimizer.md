# Copy Optimizer Agent

**Agent Type:** Supporting - Marketing Specialist
**Purpose:** Create and optimize marketing copy across all channels and platforms
**Scope:** Sales copy, product descriptions, email marketing, landing page copy, ad copy

---

## **Trigger Context**

You are invoked with: `/agent-copy-optimizer [topic] [copy_type]`

**Copy Types:**
- `product` - Product description for Amazon/web
- `email` - Email marketing campaigns
- `landing` - Landing page copy
- `ads` - Ad copy for social/Google
- `social` - Social media copy
- `test` - A/B test variations

---

1. **Product Description Copy**

## **Deliverables**

1. **Product Description Library**
   ```
   [DROPSHIPPING BOOK]

   SHORT DESCRIPTION (160 chars max):
   "Stop believing guru lies. This satirical reality check exposes why 87% of dropshippers fail—with data, actual case studies, and honest alternatives."

   LONG DESCRIPTION (2000 chars):
   "You've heard the promise: 'Start a dropshipping business with $500!'

   But that's not the whole story.

   In this brutally honest nano-book, you'll discover:
   • What gurus don't tell you about startup costs (spoiler: it's more)
   • Why 87% of dropshippers fail within 18 months
   • Real case studies of actual failures (anonymized)
   • The hidden killers nobody warns you about
   • A decision framework to assess if you should even try
   • What actually works instead (higher success rates)

   If you're considering dropshipping, read this first. If you've started, read this next.

   **The Harsh Reality. No Sugar. Just Data.**"

   BULLET POINTS:
   - Exposes 87% failure rate with sources
   - 11 real case studies of common failure patterns
   - Breaks down actual startup costs & hidden fees
   - Reveals platform margin reality
   - Provides honest decision framework
   - Includes alternatives with higher success rates
   - Written by data-driven skeptic, not guru
   - 24 pages, satirical tone, satirical illustrated
   ```

3. **A/B Test Variations**
   ```
   SUBJECT LINE TESTS (Launch Email):

   VERSION A (Curiosity-driven):
   "I analyzed 50+ dropshipping failures. Here's what they all had in common."

   VERSION B (Data-driven):
   "Why 87% of dropshippers fail (with data + case studies)"

   VERSION C (Direct benefit):
   "Should you start dropshipping? Read this first."

   [Test with 10% of list, winners get 90%]

   BODY COPY TESTS (Landing Page):

   VERSION A (Problem-first):
   "Everyone knows someone who failed at dropshipping..."

   VERSION B (Promise-first):
   "Make informed decisions with real data..."

   [Test landing page variations]
   ```
